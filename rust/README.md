# nitor-vault

## Build

```console
cargo build
cargo build --release
cargo run
```

## Format code

Using [rustfmt](https://github.com/rust-lang/rustfmt)

```console
cargo fmt
```

## Lint

Using [Clippy](https://github.com/rust-lang/rust-clippy)

```console
cargo clippy
cargo clippy --fix
```

## Update dependencies

```console
cargo update
```
