# nitor-vault

CLI and library for encrypting keys and values using client-side encryption with AWS KMS keys.

## Build

```shell
# debug
cargo build
cargo run
# release
cargo build --release
cargo run --release
```

## Format code

Using [rustfmt](https://github.com/rust-lang/rustfmt)

```shell
cargo fmt
```

## Lint

Using [Clippy](https://github.com/rust-lang/rust-clippy)

```shell
cargo clippy
cargo clippy --fix
```

## Update dependencies

```shell
cargo update
```
